# Generated Results Bundle

This folder collects the artifacts generated while running scripts `00` through `13`.

Included here:

- `stage_results/`: evaluation outputs and reports from `CENG493_Project/results`
- `processed_data/`: generated JSONL training/evaluation data that fits GitHub limits
- `index/`: FAISS index and metadata generated by script `02`
- `model_configs/`: lightweight model configs plus the Qwen LoRA adapter
- `MANIFEST.csv`: file list with sizes and timestamps
- `LARGE_ARTIFACTS_NOT_PUSHED.md`: files that exist locally but are too large for normal GitHub push

Large model/checkpoint files were not copied here because GitHub blocks files over 100 MiB and the largest local artifacts are several GB.
